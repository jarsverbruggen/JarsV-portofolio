## Overview

Proyek ini melakukan analisis dataset **UCI Heart Disease** untuk memprediksi kemungkinan penyakit jantung menggunakan beberapa algoritma machine learning, yaitu Logistic Regression, Decision Tree, Random Forest, dan Multi-Layer Perceptron (MLP).

## Key Steps

- **Preprocessing Data**: Pembersihan data, penanganan nilai yang hilang, serta encoding fitur kategorikal.
- **Exploratory Data Analysis (EDA)**: Eksplorasi distribusi data dan korelasi antar fitur.
- **Feature Selection**: Pemilihan fitur paling relevan untuk meningkatkan performa model.
- **Dimensionality Reduction**: Reduksi dimensi untuk menyederhanakan ruang fitur.
- **Model Evaluation**: Evaluasi performa setiap model menggunakan metrik yang sesuai.

## Technologies Used

- **Python** untuk implementasi dan analisis.
- **Logistic Regression, Decision Tree, Random Forest, MLP** sebagai algoritma pembanding.

## Outcome

Proyek ini memberikan pemahaman menyeluruh tentang alur kerja machine learning, mulai dari pengolahan data mentah hingga membandingkan performa beberapa model untuk menentukan pendekatan terbaik.
